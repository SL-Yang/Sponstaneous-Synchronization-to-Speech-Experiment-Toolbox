Welcome to the sss task operation document


In hallucination estimation, you can test a subject's tendency to hear hallucinations

In the sss outcome, you can see statistics for all subjects

In instructions_slides and instructions, you can edit instructions

In sounds, you can view the sounds used in the experiment

In results, you can view and save data from your experiment

In protocol, you can see the basic flow of an experiment





Run experiment.py to enter the test interface

Run compute_plv.py to calculate plv

Running praat allows recording (16000HZ)
